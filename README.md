🚨 Problem Statement

Today, millions of people spend more than 8–12 hours daily in front of laptops and screens.

This leads to:

👀 Eye strain
🪑 Poor posture
🤕 Headaches
🧠 Mental fatigue
📉 Loss of concentration
❌ Unhealthy digital habits

Most productivity applications focus only on increasing efficiency but completely ignore user health and wellness.

👉 Drustii AI aims to solve this problem.

💡 Solution

Drustii AI uses real-time computer vision and wellness analytics to help users maintain healthy digital habits while improving productivity.

The system continuously monitors:

📏 Screen distance
🪑 Posture
👁️ Blink rate
⏳ Focus sessions
📈 Productivity patterns

and provides real-time alerts and wellness guidance.
